Game Matcher

React App Flask App

Game Matcher es una aplicación web full-stack que permite a los usuarios descubrir y gestionar videojuegos personalizados según sus preferencias. La aplicación combina una base de datos local con la API externa de RAWG.

Características principales

  • Sistema de Recomendaciones Personalizadas: Basado en géneros y plataformas preferidas
  • Gestión de Favoritos: Los usuarios pueden marcar juegos como favoritos o no favoritos
  • Base de Datos Dual: Combina juegos locales con datos de RAWG API
  • Onboarding Intuitivo: Proceso guiado de 4 pasos para nuevos usuarios
  • Panel de Administración: Gestión completa de usuarios, juegos y configuraciones
  • Autenticación Robusta: Sistema de roles (usuarios/administradores)

Tecnologías utilizadas

React 18 Vite React Router Bootstrap 5 FontAwesome Context API Flask SQLAlchemy JWT CORS Alembic PostgreSQL Render
