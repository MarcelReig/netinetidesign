Porfolio/ CODIGO NUEVO

Entre 2017 y 2019 tengo la oportunidad de colaborar con el equipo de CODIGONUEVO ayudando en la realización de una serie de documentales web para El Ayuntamiento de Barcelona, el Periódico y la AMB.

La tecnología utilizada es el generador de sitios web estáticos ZURB Template, y JavaScript, haciendo uso de las librerías Vissense y Slick.
